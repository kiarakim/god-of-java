### 클래스는 상태(State)와 행위(Behavior)가 있어야 한다.

예를들어 핸드폰의 상태엔

- 밝기가 최대인 상태

- 배터리가 10% 남아있는 상태

- 폰트가 궁서체인 상태

등이 있고

핸드폰의 행위엔

- 밝기를 낮추는 행위

- 배터리를 충전하는 행위

- 폰트를 변경하는 행위 

등이 있다. 

이 내용을 자바로 표현해보자.

``` java
public class Phone {
	public Phone() {
    }
}
```

Phone 클래스 선언문 아래 메소드같이 생긴 것을 생성자(Constructor)이라고 부른다. 

Phone 클래스 객체를 생성할 때 필요한데, 이 클래스의 밝기 상태, 배터리 상태, 폰트 크기가 최소인 상태 등을 다음과 같은 변수로 나타낼 수 있다.
